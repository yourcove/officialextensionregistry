# Notification Settings

A Cove extension that adds notification preferences to the settings panel.

## Features

- **Browser Notifications**: Get notified when scans complete, metadata is updated, etc.
- **Configurable Alerts**: Choose which events trigger notifications
- **Quiet Hours**: Set time ranges when notifications are suppressed
- **Notification History**: View past notifications

## Installation

Install from the Cove extension registry in Settings → Extensions → Find and Install Extensions.
