# Scene Analytics

A Cove extension that adds play count tracking and an analytics tab to scene detail pages.

## Features

- **Play Count Tracking**: Automatically tracks how many times each scene has been viewed
- **Viewing History**: Timeline of when scenes were watched
- **Statistics Dashboard**: Analytics tab on scene detail pages showing viewing patterns
- **Rating Distribution**: Visual breakdown of how you've rated scenes

## Installation

Install from the Cove extension registry in Settings → Extensions → Find and Install Extensions.
