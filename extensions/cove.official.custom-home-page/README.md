# Custom Home Page

A Cove extension that replaces the default home page with an enhanced dashboard.

## Features

- **Recent Activity**: Shows recently added and viewed scenes, performers, and studios
- **Library Statistics**: Overview of your library size, file counts, and storage usage
- **Quick Actions**: One-click buttons for common operations like scanning, generating thumbnails, etc.
- **Customizable Layout**: Configure which widgets appear on your dashboard

## Installation

Install from the Cove extension registry in Settings → Extensions → Find and Install Extensions.

## Configuration

After installation, the extension's settings panel will appear in Settings → Extensions where you can customize the dashboard layout and which widgets are displayed.
